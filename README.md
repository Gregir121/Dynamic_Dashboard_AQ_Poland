☁️Dynamic Dashboard for Air Quality in Poland - Constructed in R shiny dynamic dashboard contains of dynamic API download and data handling.

🛠 Tech Stack

Language: R

Optimization: Shiny, DT, tidyverse

Visualization: ggplot2, plotly, leaflet, shinycssloaders

API Packages: httr, jsonlite

Environment: Local env

📂 Project Structure

R/ - Api functions for dynamic download

tests/ - Main testing logic 

main branch - Main code including UI and server functions

🚀 Quick Start

- Clone repo: git clone https://github.com/Gregir121/Dynamic_Dashboard_AQ_Poland.git
- Open: Rstudio
- Dependencies: install.packages(c("shiny", "tidyverse", "leaflet", "plotly", "httr", "jsonlite", "DT", "shinycssloaders"))
- Run app: shiny::runApp()


🔗 Live Demo: [https://gregir121.shinyapps.io/gios-air-quality-dashboard/](https://gregir121.shinyapps.io/gios-air-quality-dashboard/)
